# Database Engineering Study

This repository contains notes, exercises, and projects related to the Database Engineering course. It covers fundamental and advanced concepts of how databases work under the hood.

## 📚 Topics Covered

- **Database Internals**: Storage engines, indexing (B-Trees, LSM Trees), and page management.
- **Concurrency Control**: Transactions, ACID properties, and isolation levels.
- **Networking & Protocols**: How clients communicate with databases (TCP/UDP, HTTP, gRPC).
- **Performance Tuning**: Query optimization, execution plans, and caching strategies.
- **Architectures**: Replication, Sharding, Partitioning, and High Availability.
- **Engines**: Deep dives into PostgreSQL, MySQL, SQLite, and NoSQL variants.

## 🛠 Technologies Used

- **Languages**: Node.js, Python, or Go (depending on the module).
- **Databases**: PostgreSQL, Redis, MongoDB.
- **Tools**: Docker, Wireshark (for protocol analysis).

## 🚀 Getting Started

1. Clone the repository.
2. Navigate to specific module folders to find implementation examples.
3. Follow the `README.md` within each subdirectory for specific setup instructions.
